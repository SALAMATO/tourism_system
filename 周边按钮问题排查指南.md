# 周边按钮无法显示城市问题排查指南

## 问题描述
点击"周边"按钮后，无法显示周边城市列表。

## 已修复的问题

### 1. logger未定义错误
**位置**: `api/views.py`
**问题**: `nearby_by_ip` 方法中使用了 `logger.error()` 但 logger 在文件末尾才定义
**修复**: 将 `import logging` 和 `logger = logging.getLogger(__name__)` 移到文件开头

## 可能的原因及解决方案

### 原因1: 数据库中没有标记为"nearby"推荐类型的目的地

**检查方法**:
```bash
# 进入Django shell
python manage.py dbshell

# 执行SQL查询
SELECT COUNT(*) FROM destinations WHERE JSON_CONTAINS(recommendation_type, '"nearby"');
```

**如果返回0，说明没有目的地标记为nearby**

**解决方法**:
运行提供的SQL脚本 `check_nearby_destinations.sql` 中的第4条语句，为热门目的地添加nearby标记：

```sql
UPDATE destinations 
SET recommendation_type = JSON_ARRAY_APPEND(
    IF(recommendation_type IS NULL OR JSON_LENGTH(recommendation_type) = 0, 
       JSON_ARRAY('default'), 
       recommendation_type),
    '$', 'nearby'
)
WHERE is_domestic = TRUE 
  AND is_hot = TRUE
  AND NOT JSON_CONTAINS(recommendation_type, '"nearby"')
LIMIT 10;
```

或者在管理后台手动编辑目的地，勾选"IP周边推荐"选项。

### 原因2: IP定位API失败

**检查方法**:
打开浏览器控制台（F12），点击"周边"按钮，查看Console输出：
- 如果看到 "IP定位失败: xxx"，说明API调用出错
- 如果看到 "周边推荐API响应:" 但没有destinations数据，说明API返回异常

**可能的问题**:
- 高德地图API密钥无效或过期
- 网络连接问题
- API请求超时

**解决方法**:
1. 检查 `api/views.py` 第246行的高德地图API密钥是否有效
2. 尝试更换API密钥
3. 检查服务器网络是否能访问高德地图API

### 原因3: 前端JavaScript错误

**检查方法**:
打开浏览器控制台，查看是否有JavaScript错误

**常见错误**:
- `escapeHtml is not defined` - 工具函数未加载
- `api.request is not a function` - API对象未正确初始化

**解决方法**:
确保以下脚本按顺序加载：
```html
<script src="{% static 'js/utils.js' %}"></script>
<script src="{% static 'js/api.js' %}"></script>
<script src="{% static 'js/destinations.js' %}"></script>
```

### 原因4: state字段为空

**问题**: 国内目的地的state字段可能为空，导致城市筛选时无法正确分组

**检查方法**:
```sql
SELECT id, name, city, state FROM destinations WHERE is_domestic = TRUE AND state IS NULL;
```

**解决方法**:
批量更新state字段：
```sql
-- 示例：根据city推断state（需要根据实际情况调整）
UPDATE destinations SET state = '广东' WHERE city IN ('广州', '深圳', '珠海');
UPDATE destinations SET state = '北京' WHERE city = '北京';
-- ... 更多城市映射
```

## 快速修复方案

如果数据库中没有任何nearby标记的目的地，执行以下命令：

```bash
# 进入Django shell
python manage.py shell
```

```python
from api.models import Destination

# 为所有国内目的地添加nearby标记
destinations = Destination.objects.filter(is_domestic=True)
for dest in destinations:
    if 'nearby' not in dest.recommendation_type:
        dest.recommendation_type.append('nearby')
        dest.save()
    print(f"Updated: {dest.name}")

print(f"Total updated: {destinations.count()}")
```
